这里主要涉及三个关键注解：

  • @ExceptionHandler，指明被注解方法是对哪一类异常进行处理（单独使用，仅可处理所在Controller中的异常)
  • @ControllerAdvice，其实与Controller这类注解类似，也是一个@Component，其与ExceptionHandler配合使用可以在任意位置进行异常处理。
  • @ResponseStatus，指定返回 HTTP 状态码
